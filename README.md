Smart OLED MacroPad

A customizable MacroPad with an OLED display, designed to provide quick access to essential travel and utility tools.

## Features
Google Maps launcher,
Weather information,
Currency converter,
Travel checklist,
OLED display interface,
One-key access to commonly used functions,
Plug-and-play operation,
The customize of buttons is released soon with the help of qmk software.
The oled display shows the time.

## software
kicad
fusion360
qmk

## Parts(or)bom
SEED XIAO board,
4 mx switches,
1 oled 0.91" screen
